
### 题目描述：

输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

### 分析：
如果左边还有1， 那么n-1之后和n按位与会将这个1变成0， 也就是说每个不是最左边的1都可以做一次这样的操作。比如：1100-1=1001， 1001 & 1100=1000.

  public class Solution {
      public int NumberOf1(int n) {
          if(n==0)
              return 0;
          int res =0;
          while((n&(n-1))!=0){
              res++;
              n=n&(n-1);
          }
          res++;
          return res;
      }
  }
